# 📖 Introduction

Artificial Intelligence is rapidly evolving from **LLMs (Large Language Models)** into **AI Agents** capable of planning, reasoning, calling external tools, and autonomously completing complex tasks.

This repository provides a **beginner-friendly yet practical** guide to help developers understand:

- 🧠 What is an AI Agent?
- 🤖 Claude Code architecture
- 🔥 DeepSeek integration
- ⚙️ Claude Code deployment
- 🛠️ CC Switch configuration
- 🚀 Real-world coding workflow

Whether you're a developer, student, or cybersecurity enthusiast, this guide will help you quickly get started with modern AI coding agents.

# 🚀 Quick Start

## ① Install Node.js

```bash
node -v
npm -v
```

---

## ② Install Git

```bash
git --version
```

---

## ③ Install Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

---

## ④ Launch Claude Code

```bash
claude
```

---

## ⑤ Connect DeepSeek

Configure your API in **CC Switch**, then execute:

```text
/model
```

Verify the connected model.

# 📊 LLM vs AI Agent

| Feature | LLM | AI Agent |
|----------|-----|----------|
| Understand Language | ✅ | ✅ |
| Plan Tasks | ❌ | ✅ |
| Call Tools | ❌ | ✅ |
| Modify Files | ❌ | ✅ |
| Execute Commands | ❌ | ✅ |
| Long-term Memory | Limited | ✅ |
| Autonomous Workflow | ❌ | ✅ |
